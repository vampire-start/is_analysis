# 接口：getCourses  [返回](../README.md)
用例： [课程列表](../用例/课程列表.md)
- 接口名称：getCourses
    
- 功能：
    返回该用户所存在课程的课程列表。   
    
- API请求地址： 
    /v1/api/getCourses/{term_id}/{user_id}

- 请求方式 ：
    GET  

- 请求参数说明:        
    |参数名称|说明|
      |:---------:|:--------------------------------------------------------|
      |term_id|学期id|
      |user_id|用户id|
    
- 返回实例：

        {
            "status": true,
            "courses":[{
                "id":1,
                "name":"信息系统分析与设计",
                "desc":"一门有趣的课程"
            },
            {
            ...其他课程
            }
            ]
        }
  
- 返回参数说明：    
 
  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|      
  |status|bool类型，true表示正确的返回，false表示有错误|
  |courses|数组类型，元素表示返回的每个课程的信息|
